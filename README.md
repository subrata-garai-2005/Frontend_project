# 🌐 Digital Agency Website | HTML, CSS, JavaScript

A modern and responsive **digital agency website** built using **HTML, CSS, and JavaScript**, designed to showcase web services, portfolio projects, and client engagement. The project features a clean UI, smooth animations, and is deployed on Render for live access.

---

## ✨ Overview

This project is a fully responsive web services website suitable for freelancers, startups, and digital agencies. It highlights services like web development, UI/UX design, and branding through a visually appealing layout and interactive components.

The design focuses on a modern dark theme with gradient elements, structured sections, and clear call-to-action buttons to improve user engagement and conversion.

---

## 🚀 Features

- 🎨 Modern and clean UI design  
- 📱 Fully responsive layout (mobile-friendly)  
- ⚡ Smooth scrolling and animations  
- 🧭 Sticky navigation bar  
- 💼 Services showcase section  
- 📊 Stats and achievements section  
- 📬 Contact form (EmailJS supported)  
- 🌐 Live deployment on Render  

---

## 🛠️ Tech Stack

- HTML – Structure  
- CSS – Styling (Flexbox, Grid, Animations)  
- JavaScript – Interactivity
